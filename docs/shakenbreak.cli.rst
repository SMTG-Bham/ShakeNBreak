Commands
--------------

``ShakeNBreak`` has eight main commands: ``snb-generate``, ``snb-generate_all``, ``snb-run``,
``snb-parse``, ``snb-analyse``, ``snb-plot``, ``snb-regenerate`` and ``snb-groundstate``.
